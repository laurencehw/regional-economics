# Lab 7 Code

Scripts for the services trade gravity lab. Planned:

- `gravity_services_scaffold.py` — PPML gravity model estimation for bilateral services trade
- `stri_tariff_equivalent.py` — Convert STRI scores to ad-valorem tariff equivalents
- `servicification_decomposition.py` — TiVA-based decomposition of service content in manufacturing exports
- `cloud_geography_mapper.py` — Map cloud data center regions against data localization regimes
- `fetch_wto_services_trade.py` — Fetch bilateral services trade from WTO BOP statistics
- `fetch_oecd_stri.py` — Fetch OECD STRI scores
