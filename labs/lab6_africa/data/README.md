Store Lab 6 raw, mapped, and intermediate data here.

Key files:
- `source_mappings.json`: column mappings for VIIRS, Afrobarometer, and adjacency edge lists.
- `raw_templates/`: minimum reproducible template files used in smoke tests and quick-start runs.
- `panel_mapped.csv`: canonical mapped panel produced by `code/prepare_lab6_inputs.py`.
- `adjacency_mapped.csv`: canonical spatial links produced by `code/prepare_lab6_inputs.py`.
